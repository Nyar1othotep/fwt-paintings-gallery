## Локальный запуск

1. Копировать `.env.example` в `.env`
2. В терминале:

   ```
   npm install
   npm start
   ```

3. Проверка ESLint:

   ```
   npm run lint
   ```

4. Визуализатор модулей:

   ```
   npm run build
   ```

   Появится файл **"analyse.html"**.
